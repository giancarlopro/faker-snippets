# Faker Snippets

VSCode extension with code snippets for [Faker Gem](https://github.com/stympy/faker)

## Installation
1. Go to extensions. ( `ctrl + shift + x`)
2. Search for "Faker Snippets"
3. Click `install`
4. Click `reload`
